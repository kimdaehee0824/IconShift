# IconShift

> [English](README.md), [한국어](README_ko.md), [日本語](README_jp.md)

IconShiftは、macOSのライト・ダークモードに合わせてアプリアイコンを切り替える小さなメニューバーアプリです。アプリごとにライト用とダーク用のアイコンを指定すると、システムの外観が変わるたびにFinderとDockのアイコンも切り替わります。

1つの背景だけを想定して作られたSafariやChromeのWebアプリアイコンを使っている場合に特に便利です。

## 主な機能

- **自動切り替え** — 現在のシステム外観に合うライトまたはダークアイコンを使用します。
- **アプリごとの設定** — システムに従うほか、「常にライト」または「常にダーク」に固定できます。
- **簡単なアイコン設定** — PNGまたはICNSファイルを各アイコン欄にドロップするだけです。
- **ログイン時に起動** — サインイン後に静かに起動し、アイコンを同期した状態に保ちます。
- **メニューバーアイコンの表示切り替え** — アイコンを隠してもIconShiftはバックグラウンドで動作し続けます。
- **PWAの復元** — ブラウザのアップデートでリセットされたアイコンも、IconShiftの起動時に再適用します。

## 動作環境

| | 要件 |
| --- | --- |
| macOS | 14以降 |
| Mac | AppleシリコンまたはIntel |

IconShiftを使うためにXcodeや開発者ツールを用意する必要はありません。

## インストール

1. [GitHub Releases](https://github.com/kimdaehee0824/IconShift/releases)から最新の`IconShift-<バージョン>.zip`をダウンロードします。
2. ZIPを展開し、`IconShift.app`を「アプリケーション」フォルダに移動します。
3. IconShiftを一度開きます。現在のリリースはad-hoc署名を使用しているため、macOSが初回起動をブロックすることがあります。アプリを開こうとしたあと、**システム設定 > プライバシーとセキュリティ**を開き、**セキュリティ**で**このまま開く**をクリックしてから、もう一度**開く**を確認します。
4. IconShiftが初めてアプリアイコンを変更するときに、**アプリ管理**へのアクセスを許可します。通知を閉じてしまった場合は、**システム設定 > プライバシーとセキュリティ > アプリ管理**でIconShiftを有効にします。

Developer ID署名と公証に対応するまでは、IconShiftのアップデート後にセキュリティまたはアプリ管理の許可をやり直す必要がある場合があります。

## 使い方

1. IconShiftを開き、サイドバーの**アプリを追加**をクリックします。
2. インストール済みのアプリを選び、**ライトアイコン**と**ダークアイコン**の欄に画像をドロップします。
3. 自動切り替えには**システムの外観に従う**をオンにしておきます。オフにすると、**常にライト**または**常にダーク**を選べます。
4. 選択したアイコンをすぐに再適用したいときは、**今すぐ適用**をクリックします。

**設定 > 一般**では、**ログイン時に起動**と**メニューバーアイコンを表示**を変更できます。メニューバーアイコンを隠したあとでウインドウを開くには、IconShiftをもう一度起動してください。**設定 > 情報**では、インストール済みのバージョンとライセンスを確認できます。

Finderのアイコンはすぐに更新されます。対象のアプリがすでに起動している場合、Dockにはそのアプリを終了して開き直すまで以前のアイコンが残ります。これはmacOSの通常の動作です。IconShiftは起動時に設定済みのアイコンを再適用するため、SafariやChromeのアップデート後にPWAアイコンを復元する場合にも役立ちます。
